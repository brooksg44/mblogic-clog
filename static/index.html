Quick Start

Load the web module and start the server with your PLC program:

;; Load the web interface module
(ql:quickload :mblogic-cl/web)

;; Compile your IL program
(defparameter *program* (mblogic-cl:compile-il-file "test/plcprog.txt"))

;; Create an interpreter
(defparameter *interp* (mblogic-cl:make-plc-interpreter :program *program*))

;; Start the web server
(mblogic-cl-web:start-web-server :port 8080 :interpreter *interp*)

;; Or use the quick-start function:
(mblogic-cl-web:quick-start "test/plcprog.txt" :port 8080)

API Endpoints

  • GET /api/statistics - PLC runtime statistics
  • GET /api/data?addr=X1,Y1,DS1 - Read address values
  • GET /api/program?subrname=main - Get ladder structure
  • GET /api/subroutines - List subroutines
  • POST /api/control/start - Start PLC execution
  • POST /api/control/stop - Stop PLC execution
  • POST /api/control/step - Execute single scan

About

MBLogic-CL is a Common Lisp implementation of a soft PLC that compiles and executes Instruction List (IL) programs. This web interface provides ladder diagram visualization and live monitoring capabilities.

Based on the original MBLogic Python system for industrial automation.
